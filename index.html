3 צעדים פשוטים לחיבור הנתונים
1
צור פרויקט Supabase חינמי
Supabase הוא שירות מסד נתונים חינמי שיאחסן ויסנכרן את הנתונים שלך
🌐 פתח supabase.com ←
לחץ Start your project → הירשם עם Google → New project → בחר שם ← בחר סיסמה חזקה → Create project
2
הרץ את קוד ה-SQL הבא
בפרויקט שלך: לחץ על SQL Editor בתפריט → New query → הדבק ולחץ Run
create table if not exists user_data ( id uuid default gen_random_uuid() primary key, user_id uuid references auth.users(id) on delete cascade unique not null, data jsonb default '{}' not null, updated_at timestamptz default now() not null ); alter table user_data enable row level security; create policy "users_own_data" on user_data for all to authenticated using (auth.uid() = user_id) with check (auth.uid() = user_id);
3
הכנס את מפתחות הפרויקט
בפרויקט Supabase: Project Settings → API → העתק את ה-URL וה-anon key
